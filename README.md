**html-site-template**

A simple HTML & CSS template to quickly start a webpage with easy to customize webpage templates.

## Copy templates

1. Clone the repository to your local machine or press the *green code* button.

   ```bash
   git clone https://github.com/doggodgcodes/html-site-template.git
   cd html-site-template
   ```
